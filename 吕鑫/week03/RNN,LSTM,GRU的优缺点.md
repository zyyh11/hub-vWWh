# RNN,LSTM,GRU优缺点：
## 原始LSTM:
### 优点
- **解决了RNN梯度消失的问题，**
- **loss变化斜率平缓**
### 缺点
- **训练耗时较长**
- 单向传递消息
- loss变化较慢，epochs越大，loss越小

## RNN
### 优点
- **结构简单**：容易理解和实现,数据链式传递。
- **参数最少**
### 缺点
- **梯度消失/爆炸**：
  - 信息在传递过程中指数级衰减
  - 读到句子末尾时，开头的信息几乎完全丢失
  - 实验中Loss始终在2.35左右，和随机猜测一样无法学习
- **作用极少**

## 双向多层LSTM
### 优点
- 双向校验数据，信息传递准确率高，epochs=4，loss= 0.2349
- 防止过拟合
- 适合精细度要求过高的场景

### 缺点
- 训练时间最长
- 参数过多
- 不适合简单任务，浪费资源

## GRU
### 优点
- 结构相对简单
- 参数较少
- 准确率高
- loss下降速率很快，epochs很小的情况下就能保证loss很低
- 适合简单任务场景

### 缺点
- 训练时间较慢
- 参数少，某些信息比LSTM精细度会低
- 不适合精细度要求过高的场景
